## 项目背景

**Codex Free API Pool** 是一个免费API池服务，通过自动化账号创建+反代技术，构建一个可用的免费 Codex API 服务。

### 核心功能
1. 账号自动注册（域名池+接码服务）
2. 标准OpenAI API格式反代服务
3. 多账号轮询调度
4. 健康监控与自动替换
5. 管理后台

### 用户痛点
- Codex官方API在部分地区访问受限
- 官方定价对高频使用者成本较高
- 单个账号有配额限制
- 账号管理复杂
- 缺乏统一API入口
