## WHAT (用户需求)

### 核心功能需求

#### 1. 影院浏览与选择 (P0)
- **功能**: 用户能够查看按优先级排序的影院列表
- **场景**: 打开小程序 → 看到推荐影院 → 选择心仪影院
- **关键要素**:
  - 影院按推荐权重排序（利润率、成功率、出票速度）
  - 标签系统：店长直供、今日特价、秒出票、外部调票
  - 低优先级影院需弹窗警告

#### 2. 排期查询 (P0)
- **功能**: 查看影院当日排期和场次详情
- **场景**: 进入影院详情 → 查看电影列表 → 选择场次
- **关键要素**:
  - 显示电影名称、海报、评分
  - 场次时间、影厅、价格
  - 场次剩余座位情况

#### 3. 下单与支付 (P0)
- **功能**: 选择座位、填写订单、微信支付
- **场景**: 选场次 → 选座位偏好 → 确认支付 → 等待出票
- **关键要素**:
  - 座位偏好选择（靠前排/靠后排/中间位置）
  - 订单30分钟支付超时
  - 微信支付集成

#### 4. 订单状态跟踪 (P0)
- **功能**: 实时查看订单状态和处理进度
- **场景**: 支付成功 → 查看待接单 → 查看出票进度 → 获取取票码
- **关键要素**:
  - 状态流转：pending → paid → accepted → ticketed → completed
  - 实时通知（飞书/微信推送）
  - 暗号系统（出票后显示）

#### 5. 管理员接单出票 (P0)
- **功能**: 后台管理订单、选择出票渠道、录入座位号
- **场景**: 收到新订单通知 → 查看订单详情 → 接单 → 选择渠道 → 录入信息
- **关键要素**:
  - 多渠道出票选择（影院经理/API/自己找券）
  - 一键复制话术发送微信
  - 拒绝订单并自动退款+补偿券

#### 6. 暗号管理 (P1)
- **功能**: 系统生成和管理取票暗号
- **场景**: 已出票状态 → 显示取票暗号 → 开场后2小时降级
- **关键要素**:
  - 祝福语暗号（财源广进、平安喜乐）
  - 数字暗号（888、666、520）
  - 生命周期管理（隐藏→高亮→降级）

## WHY (用户痛点)

### 用户端痛点

| 痛点 | 严重程度 | 当前问题 | 期望结果 |
|------|---------|---------|---------|
| **购票体验差** | 高 | 传统购票流程繁琐，影院信息不透明 | 智能推荐，一键下单 |
| **出票时间长** | 高 | 外部调票等待时间长，用户体验差 | 分级提示，合理预期 |
| **取票流程复杂** | 中 | 需要排队取票，容易丢失电子票 | 暗号系统，快速取票 |
| **订单状态不透明** | 中 | 不知道订单处理进度 | 实时状态更新，推送通知 |
| **退款流程慢** | 中 | 订单拒绝后退款周期长 | 即时退款+补偿券 |

### 管理员端痛点

| 痛点 | 严重程度 | 当前问题 | 期望结果 |
|------|---------|---------|---------|
| **多渠道管理难** | 高 | 影院经理、API、自己找券多渠道切换 | 统一界面，快速选择 |
| **订单处理效率低** | 高 | 手动录入信息，沟通成本高 | 一键复制话术，快速出票 |
| **运营状态切换麻烦** | 中 | 爆单时无法快速暂停接单 | 一键切换运营状态 |

## 使用场景详细说明

### 场景A: 主流观影流程（80%用户）

```
用户打开小程序
    ↓
首页显示今日排期概览（按推荐排序）
    ↓
点击推荐影院 → 查看影院详情和排期
    ↓
选择电影场次 → 进入下单页
    ↓
选择座位偏好、票数、备注
    ↓
确认支付 → 微信支付 → 支付成功
    ↓
跳转订单详情 → 查看待接单状态
    ↓
管理员接单 → 出票 → 推送通知
    ↓
用户查看取票暗号 → 到店取票观影
```

**边界情况**:
- 用户未在30分钟内支付 → 订单自动取消
- 管理员拒绝订单 → 自动退款+补偿券
- 开场前2小时仍未出票 → 紧急提醒管理员

### 场景B: 低优先级影院选择（20%用户）

```
用户点击外部调票影院
    ↓
弹窗警告："该影院为外部调票，出票时长约15-30分钟"
    ↓
用户选择：返回推荐影院 / 继续下单
    ↓
如继续 → 正常下单流程（但有心理预期）
```

**设计意图**: 通过前置警告降低用户预期，减少催单和投诉。

### 场景C: 管理员后台操作流程

```
收到新订单通知（飞书/微信推送）
    ↓
打开管理后台 → 订单列表页
    ↓
点击订单卡片 → 查看订单详情
    ↓
点击【接单】→ 进入渠道选择
    ↓
选择出票渠道（影院经理/API/自己找券）
    ↓
出票成功 → 录入座位号+选择暗号
    ↓
系统推送通知用户"已出票"
```

**运营状态切换**:
- 正常接单：正常接收订单
- 人工超载（爆单）：显示公告，暂停接单
- 打烊休息：显示休息提示，不可下单
