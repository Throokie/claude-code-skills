# 电商数据分析仪表盘 - 产品需求文档 (PRD)

## 1. 文档概述

### 1.1 文档信息
- **版本**: v1.0
- **创建日期**: 2026-03-23
- **作者**: AI Assistant
- **状态**: 草稿

### 1.2 产品愿景
为电商平台运营人员提供一个直观、实时的数据监控仪表盘，帮助快速掌握销售动态和库存状况，支持数据驱动的运营决策。

---

## 2. 用户故事

### 2.1 主要用户角色
| 角色 | 描述 | 核心需求 |
|------|------|----------|
| 运营人员 | 日常监控销售数据、调整运营策略 | 实时数据、趋势分析 |
| 库存管理员 | 监控库存状态、预防缺货/积压 | 库存预警、补货建议 |
| 运营主管 | 查看整体业绩、制定战略决策 | 综合报表、对比分析 |

### 2.2 用户故事列表

#### US-001: 实时监控销售数据
**作为** 运营人员
**我想要** 在仪表盘上查看实时销售数据
**以便** 及时了解当前销售状况

**验收标准**:
- 数据更新延迟不超过 5 分钟
- 显示今日销售额、订单量、客单价
- 与昨日/上周同期数据对比

#### US-002: 查看销售趋势
**作为** 运营人员
**我想要** 查看销售趋势图表
**以便** 分析销售变化规律

**验收标准**:
- 支持日/周/月时间维度切换
- 图表类型：折线图、柱状图
- 支持同比/环比对比

#### US-003: 监控库存状态
**作为** 库存管理员
**我想要** 查看各品类库存状态
**以便** 及时补货或处理积压

**验收标准**:
- 显示库存总量、在途库存、可用库存
- 按品类/品牌/SKU 多维度查看
- 库存周转率计算

#### US-004: 库存预警提醒
**作为** 库存管理员
**我想要** 接收库存预警通知
**以便** 预防缺货或库存积压

**验收标准**:
- 低库存预警（低于安全库存线）
- 高库存预警（超过滞销阈值）
- 临期商品预警
- 支持邮件/站内信通知

#### US-005: 数据导出
**作为** 运营主管
**我想要** 导出报表数据
**以便** 进行离线分析或汇报

**验收标准**:
- 支持 Excel、CSV 格式导出
- 支持 PDF 格式导出图表
- 可导出当前筛选条件下的完整数据

### 3.2 功能详细说明

#### F-001: 仪表盘首页

**功能描述**: 提供销售数据概览和快捷入口

**核心指标卡片**:
| 指标名称 | 说明 | 数据来源 |
|---------|------|----------|
| 今日销售额 | 当日累计销售金额 | 订单系统 |
| 今日订单量 | 当日累计订单数量 | 订单系统 |
| 今日客单价 | 平均每个订单金额 | 计算: 销售额/订单量 |
| 今日转化率 | 下单访客占比 | 计算: 订单量/访客数 |
| 库存总值 | 当前库存商品价值 | 库存系统 |
| 预警商品数 | 触发预警规则的SKU数 | 预警系统 |

**实时更新机制**:
- 自动刷新间隔: 5分钟
- 手动刷新按钮
- 数据更新动画提示

#### F-002: 销售数据分析

**2.1 销售趋势分析**

| 配置项 | 选项 |
|--------|------|
| 时间维度 | 今日、近7天、近30天、近90天、自定义 |
| 对比模式 | 同比、环比、无对比 |
| 指标选择 | 销售额、订单量、访客数、转化率、客单价 |
| 图表类型 | 折线图、柱状图、面积图 |

**2.2 品类销售分析**

- 品类销售占比饼图
- 品类销售排行 TOP 10
- 品类趋势对比（多品类同时展示）

**2.3 商品销售排行**

- 销售额排行 TOP 20
- 销量排行 TOP 20
- 转化率排行 TOP 20
- 支持按品类/品牌筛选

**2.4 地域销售分析**

- 省份销售分布地图
- 城市销售排行 TOP 10
- 地域销售趋势

#### F-003: 库存数据监控

**3.1 库存总览**

| 指标名称 | 说明 |
|---------|------|
| 总SKU数 | 系统商品SKU总数 |
| 有库存SKU | 当前有可用库存的SKU数 |
| 库存商品总值 | 库存商品总成本价值 |
| 平均库存天数 | 按当前销量预估库存可售天数 |

**3.2 库存分布**

- 按品类库存分布
- 按仓库库存分布
- 库存状态分布（正常/预警/缺货）

**3.3 库存预警管理**

**预警规则配置**:
| 预警类型 | 触发条件 | 默认阈值 |
|---------|---------|---------|
| 低库存预警 | 可用库存 ≤ 安全库存 | 安全库存=7天销量 |
| 滞销预警 | 库存天数 ≥ 滞销阈值 | 90天 |
| 缺货预警 | 可用库存 = 0 | 即时 |
| 临期预警 | 保质期剩余 ≤ 30天 | 30天 |

**预警列表功能**:
- 预警商品列表展示
- 一键生成补货建议
- 标记已处理/忽略
- 预警历史查询

#### F-004: 报表中心

**4.1 自定义报表**

- 拖拽式报表配置
- 支持多维度组合
- 保存报表模板
- 定时自动生成

**4.2 数据导出**

| 导出格式 | 支持内容 | 文件大小限制 |
|---------|---------|-------------|
| Excel | 完整数据+图表 | 50MB |
| CSV | 原始数据 | 100MB |
| PDF | 可视化报表 | 20MB |

## 5. 数据需求

### 5.1 数据源

| 数据类型 | 来源系统 | 同步方式 | 同步频率 |
|---------|---------|---------|---------|
| 订单数据 | 订单系统 | API推送 | 实时 |
| 商品数据 | 商品中心 | 定时同步 | 5分钟 |
| 库存数据 | 仓储系统 | API推送 | 实时 |
| 用户数据 | 用户中心 | 定时同步 | 15分钟 |
| 财务数据 | 财务系统 | 定时同步 | 每日凌晨 |

### 5.2 数据模型

#### 订单事实表 (fact_orders)
| 字段名 | 类型 | 说明 |
|--------|------|------|
| order_id | VARCHAR(32) | 订单ID |
| order_date | DATETIME | 下单时间 |
| user_id | VARCHAR(32) | 用户ID |
| sku_id | VARCHAR(32) | SKU ID |
| category_id | VARCHAR(32) | 品类ID |
| brand_id | VARCHAR(32) | 品牌ID |
| quantity | INT | 数量 |
| amount | DECIMAL(16,2) | 金额 |
| province | VARCHAR(32) | 省份 |
| city | VARCHAR(32) | 城市 |
| status | INT | 订单状态 |

#### 库存事实表 (fact_inventory)
| 字段名 | 类型 | 说明 |
|--------|------|------|
| sku_id | VARCHAR(32) | SKU ID |
| warehouse_id | VARCHAR(32) | 仓库ID |
| stock_quantity | INT | 库存数量 |
| available_quantity | INT | 可用数量 |
| reserved_quantity | INT | 预留数量 |
| update_time | DATETIME | 更新时间 |
